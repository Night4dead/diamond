# Diamond Kata

The goal is to make a program that takes as input a letter and draws a diamon starting with A and has the input letter at the widest point.

example: 
> ```bash
>   ./diamond d
>```

result: 
> ```bash
>     A
>    B B
>   C   C
>  D     D
>   C   C
>    B B
>     A
> ```
